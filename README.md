# Daisy's Wine Shop

#### An e-commerce wine shop site

## Description

An e-commerce web application for buying wine. 

## Deployed app

- [Daysis Wine Shop](https://daisys-wine-shop.herokuapp.com/)


## Tech stack used

- React.js
- Redux.js
- Node.js
- Express.js
- PostgreSQL
- Sequelize ORM

## Some additional technologies

- Axios
- Stripe
- passport
- passport-google-oauth
- pg
- pg-hstore
